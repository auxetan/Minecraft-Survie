SurvivalCraft Resource Pack — Custom Font Textures
====================================================

Place your custom PNG images in this folder.

REQUIRED FILE:
  eclat.png
    - Size: 8x8 pixels (or multiples: 16x16, 32x32 for higher res)
    - Content: pixel art of a gold crystal shard / spark
    - This image is mapped to the Unicode character ✦ (U+2726)
    - Used as the in-game currency symbol throughout the plugin
    - Must be a PNG with transparency (RGBA)

The font provider is declared in:
  assets/minecraft/font/default.json

Once eclat.png is in place, the ✦ character will render as
your custom pixel art icon in all in-game chat and UI text.

Tools to create pixel art:
  - Piskel (free, browser-based): https://www.piskelapp.com/
  - LibreSprite (free, desktop):  https://libresprite.github.io/
  - Aseprite (paid, recommended): https://www.aseprite.org/
